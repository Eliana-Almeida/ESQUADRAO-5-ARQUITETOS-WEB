# ESQUADRÃO 5: OS ARQUITETOS WEB

Projeto tecnológico desenvolvido para demonstrar bibliotecas Python aplicadas ao desenvolvimento web moderno.

## Instalar
```bash
pip install -r requirements.txt
```

## Portal
```bash
python portal/app.py
```
Abrir: http://127.0.0.1:5010

## Execução das Bibliotecas

```bash
python bibliotecas/Eliana/flask_demo.py
python bibliotecas/Eliana/jinja_demo.py
```

## Conceitos
Python executa a lógica. VS Code organiza os arquivos. GitHub salva e compartilha. Terminal roda comandos. Localhost é o endereço do próprio computador. Porta identifica cada servidor.
